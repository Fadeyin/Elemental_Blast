# Полоска здоровья монстров

## Описание

Визуальный индикатор здоровья монстров, отображающийся над каждым монстром на игровом поле.

## Функциональность

### Основные характеристики

- **Расположение**: Над монстром, с отступом 2px от верхней границы
- **Размер**: Высота 4px, ширина соответствует ширине монстра
- **Компактность**: Минимальные размеры для сохранения чистоты интерфейса

### Цветовая схема

1. **Фон полоски**: Тёмно-серый (RGB: 0.2, 0.2, 0.2) с прозрачностью 0.6
2. **Текущее здоровье**: Зелёный (RGB: 0.2, 0.8, 0.2) с прозрачностью 0.9
3. **Полученный урон**: Красный (RGB: 0.9, 0.2, 0.2) с прозрачностью 0.7

### Поведение

1. **При максимальном HP**:
   - Полоска полностью зелёная
   - Занимает 100% ширины

2. **При получении урона**:
   - Зелёная часть уменьшается пропорционально оставшемуся HP
   - На месте потерянного HP появляется красная часть
   - Соотношение: `зелёная_ширина = общая_ширина * (текущий_HP / максимальный_HP)`

3. **При уничтожении монстра**:
   - Полоска исчезает вместе с монстром
   - Учитывается alpha-канал для плавного исчезновения

## Техническая реализация

### Функция отрисовки

```gdscript
func _draw_monster_health_bar(top_left: Vector2, width: float, hp: int, max_hp: int, alpha: float = 1.0)
```

### Параметры

- `top_left: Vector2` - Верхняя левая точка монстра
- `width: float` - Ширина монстра (используется как ширина полоски)
- `hp: int` - Текущее здоровье монстра
- `max_hp: int` - Максимальное здоровье монстра (initial_hp)
- `alpha: float` - Прозрачность (для анимаций появления/исчезновения)

### Интеграция

Функция вызывается в конце `_draw_enemy_monster()`, после отрисовки всех элементов монстра:

```gdscript
_draw_monster_health_bar(anim_top_left, anim_size.x, hp, initial_hp, alpha)
```

## Особенности

1. **Адаптивность**: Полоска следует за анимациями монстра (дыхание, покачивание)
2. **Прозрачность**: Поддержка alpha-канала для анимаций
3. **Универсальность**: Работает как с текстурными, так и с процедурными монстрами
4. **Производительность**: Минимальные вычисления, отрисовка простыми прямоугольниками

## Константы

Все параметры полоски здоровья вынесены в константы в начале файла `game_board.gd`:

```gdscript
const HEALTH_BAR_HEIGHT := 4.0                              # Высота полоски в пикселях
const HEALTH_BAR_MARGIN := 2.0                              # Отступ от верхней границы монстра
const HEALTH_BAR_BG_COLOR := Color(0.2, 0.2, 0.2, 0.6)      # Цвет фона
const HEALTH_BAR_HEALTH_COLOR := Color(0.2, 0.8, 0.2, 0.9)  # Цвет здоровья
const HEALTH_BAR_DAMAGE_COLOR := Color(0.9, 0.2, 0.2, 0.7)  # Цвет урона
```

Использование констант обеспечивает:
- Лёгкую настройку визуальных параметров
- Единую точку изменения для всех полосок
- Улучшенную читаемость кода

## Будущие улучшения

Потенциальные улучшения функционала:

1. **Анимация урона**: Плавный переход от зелёного к красному
2. **Числовые значения**: Отображение HP цифрами (опционально)
3. **Пульсация**: При критическом HP (< 20%)
4. **Градиенты**: Плавные переходы цветов вместо однородной заливки
5. **Обводка**: Тонкая светлая обводка для лучшей видимости на разных фонах
