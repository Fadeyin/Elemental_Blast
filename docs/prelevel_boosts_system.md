# Система предуровневых усилений (Pre-Level Boosters)

## Описание

Система предуровневых усилений позволяет игроку выбрать до 3 бонусных фишек (по одной каждого типа) перед началом уровня. Эти фишки появятся на игровом поле в начале матча вместе с обычными фишками и бонусами от Шлема Морта.

## Типы усилений

Доступны 3 типа предуровневых усилений:

| Тип      | Название | Текстура                              | Эффект на поле                |
|----------|----------|---------------------------------------|-------------------------------|
| `bomb`   | Бомба    | `Сhip_Bonus_Bomb.png`                 | Взрыв крестом (5 клеток)      |
| `arrow`  | Стрела   | `Сhip_Bonus_Arrows.png`               | Уничтожение всей строки       |
| `rainbow`| Шар      | `Сhip_Bonus_Rainbow_Ball.png`         | Уничтожение всех фишек цвета  |

## Начальный набор

При первом запуске игры каждый игрок получает:
- **3 бомбы**
- **3 стрелы**
- **3 шара**

## Использование

### Выбор на старте уровня

На экране старта уровня игрок видит:
- Иконки всех 3 типов усилений
- Счетчик доступных усилений каждого типа (например, "x3")
- Кнопки для активации усиления (при наличии зарядов)

**Правила выбора:**
- Можно выбрать МАКСИМУМ ОДНО усиление каждого типа
- Выбор усиления списывает 1 заряд из запаса игрока
- Отмена выбора возвращает заряд обратно
- Если зарядов нет (x0), кнопка неактивна

### Покупка усилений

Когда усиления заканчиваются:
- Вместо счетчика появляется кнопка "+" (зеленая, круглая)
- При нажатии открывается диалог покупки
- Стоимость: **200 золотых** за **3 заряда** одного типа
- Покупка добавляет заряды в запас игрока

## Визуальное отображение

### Диалог старта уровня

Секция "ПРЕДУРОВНЕВЫЕ УСИЛЕНИЯ":
- HBoxContainer с 3 колонками (по одной на тип)
- Каждая колонка содержит:
  - Кнопку-переключатель (toggle) с иконкой 100x100px
  - Название усиления
  - Счетчик зарядов (золотой текст)
  - Кнопку покупки "+" (если зарядов нет)

**Стили кнопок:**
- Неактивная (нет зарядов): серая, полупрозрачная
- Активная (есть заряды): темно-синяя с золотой рамкой
- Выбранная: ярко-синяя с яркой рамкой

### Спавн на поле

Выбранные предуровневые усиления:
- Спавнятся на случайных позициях в зоне игрока
- Заменяют обычные фишки
- Спавнятся ВМЕСТЕ с бонусами от Шлема Морта

## Технические детали

### Переменные в LevelManager

```gdscript
var prelevel_boosts := {
    "bomb": 3,
    "arrow": 3,
    "rainbow": 3
}

const PRELEVEL_BOOST_PURCHASE_COST := 200
const PRELEVEL_BOOST_PURCHASE_COUNT := 3
```

### Функции

```gdscript
# Получить количество зарядов
func get_prelevel_boost_count(boost_type: String) -> int

# Использовать один заряд (при выборе на старте уровня)
func use_prelevel_boost(boost_type: String) -> bool

# Купить заряды (3 шт за 200 монет)
func purchase_prelevel_boosts(boost_type: String) -> bool

# Получить текстуру для иконки
func get_prelevel_boost_texture(boost_type: String) -> Texture2D
```

### Сохранение прогресса

Данные сохраняются в `user://progress.cfg` в секции `[prelevel_boosts]`:

```ini
[prelevel_boosts]
bomb=2
arrow=3
rainbow=1
```

## Интеграция с игровым процессом

### level_start_dialog.gd

1. При создании диалога считывается количество зарядов из `LevelManager`
2. Игрок выбирает усиления через toggle-кнопки
3. При выборе вызывается `LevelManager.use_prelevel_boost(type)`
4. При отмене выбора заряд возвращается обратно
5. При нажатии "Играть" выбранные усиления передаются в `game_board.gd`

### game_board.gd

1. Сохраняет выбранные усиления в `_selected_prelevel_boosts`
2. При старте уровня вызывает `_spawn_bonus_chips_at_start()`
3. Функция собирает все бонусные фишки (предуровневые + Шлем Морта)
4. Случайно заменяет обычные фишки на бонусные

## Баланс

### Экономика

- **Стартовый набор**: 9 зарядов (3 каждого типа) - бесплатно
- **Стоимость пополнения**: 200 монет за 3 заряда одного типа
- **Цена за один заряд**: ~67 монет

Это позволяет:
- Новым игрокам попробовать систему бесплатно
- Опытным игрокам стратегически тратить монеты на нужные усиления
- Не делать усиления слишком доступными (ценность выбора)

### Игровая ценность

Предуровневые усиления:
- Помогают на сложных уровнях
- Дают стратегический выбор (какие усиления взять)
- Ограничены по количеству (нельзя взять все сразу)
- Создают ценность монет (нужно покупать заряды)
