# Система магазина

## Описание

Вкладка "Магазин" на главном экране, где игрок может приобрести пакеты монет и бустеров за реальные деньги.

## Предложения магазина

### 1. Стартовый пакет (1$) - ТОЛЬКО ОДИН РАЗ

**Содержимое**:
- 1000 золотых 🪙
- 4 бустера каждого вида 🎮
  - 4× Молоток

  - 4× Перемешивание
  - 4× Заморозка

## Технические детали

### Структуры данных

```gdscript
# LevelManager
var starter_pack_purchased: bool = false

func purchase_starter_pack():
    if not starter_pack_purchased:
        player_coins += 1000
        for type in boosters:
            booster_counts[type] += 4
        starter_pack_purchased = true
        _save_progress()

func purchase_medium_pack():
    player_coins += 2500
    for type in boosters:
        booster_counts[type] += 5
    _save_progress()

func purchase_best_pack():
    player_coins += 5000
    for type in boosters:
        booster_counts[type] += 10
    _save_progress()

func is_starter_pack_purchased() -> bool:
    return starter_pack_purchased
```

## Экономическая модель

### Ценность бустеров

**Базовая покупка**:
- 1 бустер = 350 монет

**В пакетах**:
- Стартовый: 16 бустеров + 1000 монет = ~6600 монет за 1$
- Средний: 20 бустеров + 2500 монет = ~9500 монет за 5$
- Выгодный: 40 бустеров + 5000 монет = ~19000 монет за 9$

### Цена за монету

| Пакет | Монет | Цена | Монет/$ |
|-------|-------|------|---------|
| Стартовый | 6600* | 1$ | 6600 |
| Средний | 9500* | 5$ | 1900 |
| Выгодный | 19000* | 9$ | 2111 |

*С учётом бустеров по 350 монет

### Рекомендации

**Для новых игроков**:
- Купить стартовый пакет (лучшая цена/монету)
- Даёт хороший старт (удвоение ресурсов)

**Для активных игроков**:
- Выгодный пакет (больше всего ресурсов)
- Средний пакет для пополнения

**F2P стратегия**:
- Можно играть без покупок
- Начальных 500 монет и 16 бустеров достаточно

## Монетизация

### Стратегия ценообразования

**Низкий порог входа**:
- Стартовый пакет за 1$ (очень выгодно)
- Создаёт привычку покупать

**Средний сегмент**:
- Средний пакет за 5$ (баланс)
- Для регулярных игроков

**Киты (whales)**:
- Выгодный пакет за 9$ (максимум ресурсов)
- Для хардкорных игроков

### Ожидаемая конверсия

- **Стартовый пакет**: 30-40% игроков
- **Средний пакет**: 10-15% игроков
- **Выгодный пакет**: 5-10% игроков

### Lifetime Value (LTV)

**Средний игрок**:
- Стартовый (1×): 1$
- Средний (2×): 10$
- **LTV**: ~11$

**Активный игрок**:
- Стартовый (1×): 1$
- Средний (5×): 25$
- Выгодный (2×): 18$
- **LTV**: ~44$

**Кит**:
- Все пакеты многократно
- **LTV**: 100$+

## Тестирование

### Тестовые сценарии

1. **Отображение предложений**:
   - Открыть вкладку "Магазин"
   - Проверить 3 карточки (если стартовый не куплен)
   - Проверить цвета обводок

2. **Стартовый пакет**:
   - Проверить бейдж "ТОЛЬКО 1 РАЗ!"
   - Кликнуть → диалог
   - Симуляция покупки → пакет исчезает
   - Перезапуск → пакет НЕ появляется

3. **Многоразовые пакеты**:
   - Симуляция покупки среднего
   - Проверить начисление ресурсов
   - Перезапуск → пакет всё ещё доступен

4. **Автообновление**:
   - Купить стартовый
   - Магазин обновляется автоматически
   - Счётчики в верхнем баре обновились
