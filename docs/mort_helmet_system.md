# Система "Шлем Морта" (Win Streak)

## Описание

Система "Шлем Морта" - это механика усиления игрока за серию побед без поражений. Чем больше уровней игрок проходит подряд, тем сильнее становится его шлем, давая дополнительные бонусные фишки в начале каждого уровня.

## Уровни Шлема Морта

Шлем имеет 3 уровня прокачки, которые зависят от количества побед подряд:

| Уровень | Побед подряд | Бонусные фишки в начале уровня |
|---------|--------------|-------------------------------|
| 0       | 0            | Нет бонусов                   |
| 1       | 1            | 1 стрела + 1 бомба            |
| 2       | 2            | 2 стрелы + 2 бомбы            |
| 3       | 3+           | 3 стрелы + 3 бомбы            |

## Логика работы

### Увеличение уровня

При победе на уровне:
1. Счетчик `win_streak` увеличивается на 1
2. Уровень `mort_helmet_level` обновляется автоматически на основе `win_streak`
3. Прогресс сохраняется в файл конфигурации

### Обнуление

При поражении на уровне:
1. Счетчик `win_streak` сбрасывается в 0
2. Уровень `mort_helmet_level` сбрасывается в 0
3. Игрок должен начать серию побед заново

## Визуальное отображение

### Диалог старта уровня

На экране старта уровня отображается:
- Название "ШЛЕМ МОРТА"
- 3 индикатора уровней (квадраты 1, 2, 3)
- Активные уровни подсвечиваются золотым цветом
- Неактивные уровни отображаются серым
- Под индикаторами показывается текущий бонус (например, "Бонус: 2 стрелы + 2 бомбы")

### Спавн бонусных фишек

Бонусные фишки от Шлема Морта:
- Появляются на случайных позициях в зоне игрока
- Заменяют обычные фишки в начале уровня
- Спавнятся ВМЕСТЕ с предуровневыми усилениями (если они выбраны)

## Технические детали

### Переменные в LevelManager

```gdscript
var mort_helmet_level: int = 0  # 0, 1, 2, 3
var win_streak: int = 0         # Количество побед подряд
```

### Функции

- `mark_level_completed()` - вызывается при победе, увеличивает win_streak
- `mark_level_failed()` - вызывается при поражении, обнуляет win_streak
- `get_mort_helmet_bonus_chips()` - возвращает Dictionary с количеством бонусных фишек
- `_update_mort_helmet_level()` - обновляет уровень шлема на основе win_streak

### Сохранение прогресса

Данные сохраняются в `user://progress.cfg` в секции `[mort_helmet]`:

```ini
[mort_helmet]
level=2
win_streak=2
```

## Интеграция с игровым процессом

### game_board.gd

1. При показе диалога старта уровня считываются данные из `LevelManager`
2. Игрок видит текущий уровень шлема и бонусы
3. При нажатии "Играть" бонусные фишки спавнятся на поле через `_spawn_bonus_chips_at_start()`
4. При победе вызывается `LevelManager.mark_level_completed()`
5. При поражении вызывается `LevelManager.mark_level_failed()`

## Баланс

Система мотивирует игроков играть аккуратно и стремиться к победам без поражений, давая постепенно возрастающие преимущества:

- **Уровень 1** (1 победа) - небольшая помощь для продолжения серии
- **Уровень 2** (2 победы) - заметное усиление
- **Уровень 3** (3+ побед) - максимальная награда за мастерство

Потеря серии при поражении добавляет напряжение и ценность каждой победе.
