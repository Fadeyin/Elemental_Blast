## Что было исправлено

### Проблема
Игра показывала ошибку "Ошибка загрузки игры" из-за того, что GitHub Actions **удалял весь JavaScript код** от Godot при создании HTML.

### Решение
Теперь workflow **сохраняет оригинальный код** от Godot и только добавляет мобильные мета-теги.

## 📱 Как использовать

### На телефоне

1. Откройте: https://fadeyin.github.io/Elemental_Blast/
2. Подождите загрузку (первый раз может занять ~30 секунд)
3. Играйте!

### Автоматическое обновление

Каждый раз, когда вы делаете `git push` в ветку `main`:

1. GitHub Actions автоматически соберёт игру
2. Через 3-5 минут обновления появятся на сайте
3. Обновите страницу на телефоне - готово!
